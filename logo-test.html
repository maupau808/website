Option A — Color logos on light background (current Wix)

All different colors, no visual unity.

Option B — Grayscale logos on light background

Unified but can look flat. Note logos with colored backgrounds (Toro, Scag) become gray boxes.

Option C — Grayscale + brightened logos on dark green

Probably the cleanest. Logos read as light gray silhouettes. Toro/Scag boxes become subtle gray containers.

Option D — Color logos on dark green (no grayscale)

Colorful but busy. Some logos with white backgrounds will have visible white boxes.
